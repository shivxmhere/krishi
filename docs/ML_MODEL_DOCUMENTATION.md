# ML Model Documentation

## Disease Detection
- **Architecture**: CNN (ResNet50 based)
- **Input**: 256x256 RGB Images
- **Output**: 38 Classes (Disease/Healthy)
- **Accuracy**: 95% on validation set

## Yield Prediction
- **Algorithm**: Random Forest Regressor
- **Features**: Soil, Weather, Historical Yield
- **Output**: Yield in kg/hectare
